## Icon Sizes

PatternFly icons support these size props:
- `size="sm"` - Small (16px)
- `size="md"` - Medium (18px) - default
- `size="lg"` - Large (24px)
- `size="xl"` - Extra large (32px)

## Color Guidelines

### Using the PatternFly Icon Component (Recommended)

For status icons, use the PatternFly `Icon` component from `@patternfly/react-core`:

```tsx
import { Icon } from '@patternfly/react-core';
import { CheckCircleIcon, ExclamationCircleIcon, ExclamationTriangleIcon, InfoCircleIcon } from '@patternfly/react-icons';

// Success (green)
<Icon status="success"><CheckCircleIcon /></Icon>

// Danger/Error (red)
<Icon status="danger"><ExclamationCircleIcon /></Icon>

// Warning (orange)
<Icon status="warning"><ExclamationTriangleIcon /></Icon>

// Info (blue)
<Icon status="info"><InfoCircleIcon /></Icon>

// Custom color (when status doesn't apply)
<Icon><SomeIcon color="var(--pf-v6-global--Color--100)" /></Icon>
```

**Available status values**: `success`, `danger`, `warning`, `info`, `custom`

### Direct Color Props (Alternative)

If not using the Icon wrapper, apply colors via PatternFly design tokens:

```tsx
// Success (green)
<CheckCircleIcon color="var(--pf-v6-global--success-color--100)" />

// Danger/Error (red)
<ExclamationCircleIcon color="var(--pf-v6-global--danger-color--100)" />

// Warning (gold/orange)
<ExclamationTriangleIcon color="var(--pf-v6-global--warning-color--100)" />

// Info (blue)
<InfoCircleIcon color="var(--pf-v6-global--info-color--100)" />

// Disabled (gray)
<SomeIcon color="var(--pf-v6-global--disabled-color--100)" />
```

## Visual Matching Tips

When matching icons from Figma designs:

1. **Look for shape characteristics**:
   - Circles → likely a status icon (`*CircleIcon`)
   - Triangles → likely warning (`*TriangleIcon`)
   - Arrows → navigation or action
   - Lines/bars → menu or drag handle

2. **Consider context**:
   - Top-right of modals → `TimesIcon` or `TimesCircleIcon`
   - Success messages → `CheckCircleIcon`
   - Error alerts → `ExclamationCircleIcon`
   - Buttons → action icons like `PlusIcon`, `TrashIcon`, etc.

3. **Check for variations**:
   - Many icons have both filled and outlined versions
   - Some have "Alt" variants (e.g., `PencilAltIcon`)
   - Status icons often have both `Icon` and `CircleIcon` versions

4. **Color is not distinctive**:
   - PatternFly icons are monochrome by default
   - Color is applied via CSS/props, not baked into the icon
   - Match based on shape, not color in the Figma design
