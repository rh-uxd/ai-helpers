# Evaluation Reports — GitLab Pages

Optional template for hosting self-contained HTML evaluation reports generated by
`uxd-prototype-evaluate`.

Reports are deployed via GitLab Pages on push to `main` (see `.gitlab-ci.yml` in this folder).

## Structure

```
public/
└── evals/
    ├── index.html              # Browsable report index
    ├── PROJ-298/
    │   └── index.html          # Eval report for PROJ-298
    └── ...
```

## Publishing

Configure `config/publish.yaml` (`gitlab_pages_repo`, `pages_base_url`), then run:

```bash
bash "${CLAUDE_SKILL_DIR}/scripts/publish-report.sh" .artifacts/PROJ-298/eval/
```

Re-running an eval replaces the report at the same URL.
