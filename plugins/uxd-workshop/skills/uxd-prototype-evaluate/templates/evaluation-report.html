How Usability Scores Are Calculated

INF (Inference) — fast, structural analysis from Playwright journey evidence. TA (Think-Aloud) — slow, diagnostic persona narration.

7 dimensions scored 0-3: Workflow Continuity, Cross-Persona Handoffs, Scalability, System Status, Technical Abstraction, Mental Model Fidelity, Accessibility. Dimensions marked N/A (e.g., Cross-Persona Handoffs for single-user features) are excluded — composite denominator adjusts accordingly (e.g., 18 when 1 dimension is N/A).

Patience model: starts 100%, confusion costs 5-15%, dead ends cost 10-30%, successes recover 5-10%. At 0% the persona abandons.

Prototype Evaluation Report

An automated evaluator tested this prototype against its Jira acceptance criteria using Playwright walkthroughs and simulated persona navigation.

How to read this report

Top section — Executive summary with pass/fail/flagged counts and the overall usability score.

AC Table — Each acceptance criterion with a verdict. Click View evidence to open a full-screen viewer with screenshots, think-aloud narration, and persona reactions.

Appendix tabs — Detailed evidence: persona walkthroughs, usability scores, fix history (with before/after screenshots), design compliance checks, and pipeline info.
